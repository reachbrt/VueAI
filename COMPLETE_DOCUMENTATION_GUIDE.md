## üöÄ Getting Started (3 Steps)

### Step 1: Install
```bash
npm install @aivue/core @aivue/chatbot
```

### Step 2: Configure
```bash
# .env
VITE_OPENAI_API_KEY=your-key
```

### Step 3: Use
```vue
<template>
  <AiChatWindow provider="openai" :api-key="apiKey" />
</template>

<script setup>
import { AiChatWindow } from '@aivue/chatbot';
const apiKey = import.meta.env.VITE_OPENAI_API_KEY;
</script>
```

**See:** [ALL_PACKAGES_GUIDE.md](./ALL_PACKAGES_GUIDE.md) for complete examples
